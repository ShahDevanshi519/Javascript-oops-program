# JavaScript OOPs Concepts Practice

This repository contains programs and examples of Object-Oriented Programming (OOPs) concepts implemented in JavaScript.

##  Topics Covered

### 1. Class

* Creating and using classes.
* Defining properties and methods inside a class.

### 2. Constructor

#### Default Constructor

* Constructor without parameters.
* Automatically called when an object is created.

#### Parameterized Constructor

* Constructor that accepts values during object creation.
* Used to initialize object properties.

### 3. Inheritance

#### Single Inheritance

* One child class inherits from one parent class.

#### Multilevel Inheritance

* A class inherits from another derived class.

#### Hierarchical Inheritance

* Multiple child classes inherit from the same parent class.

### 4. Polymorphism

#### Method Overloading

* Multiple methods with the same name.
* JavaScript does not support true method overloading, but it can be achieved using different arguments.

#### Method Overriding

* Child class provides its own implementation of a method inherited from the parent class.

### 5. Encapsulation

* Binding data and methods into a single unit (class).
* Data hiding using private fields.

### 6. Abstraction

* Hiding implementation details and showing only essential features.
* Achieved using abstract-like structures and methods.

### 7. Access Modifiers

* **Public**: Accessible from anywhere.
* **Private**: Accessible only within the class.
* **Protected**: Simulated using conventions in JavaScript.

### 8. Keywords Used

#### this Keyword

* Refers to the current object instance.

#### super Keyword

* Used to access parent class constructor and methods.

#### new Keyword

* Used to create objects from a class.
